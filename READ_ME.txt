ФАЙЛЫ.
  Папка exe — скомпилированные проекты (Компилятор MinGW-w64 GCC 12.2.0 64-bit, 
  Windows 10 64-bit):
     Rossler.exe  — генерирует данные для построения аттрактора Рёсслера;   
     Rossler_a.exe — генерирует данные для построения бифуркационной диаграммы 
	 при фиксированных параметрах b, c и изменении a;        
     Rossler_b.exe — генерирует данные для построения бифуркационной диаграммы 
	 при фиксированных параметрах a, c и изменении b;    
     Rossler_c.exe — генерирует данные для построения бифуркационной диаграммы 
	 при фиксированных параметрах a, b и изменении c.
При	запуске на выполнение каждого из этих файлов будут созданны файлы *.d с 
данными.    

Папка gnuplots — *.plt-файлы — скрипты для gnuplot и *.d-файлы с данными.

Папка plots — готовые графики.

Папка source — исходники программных проектов (IDE Red Panda C++ 2.1). 

ПОРЯДОК РАБОТЫ. 
1) Устанавливаем gnuplot http://www.gnuplot.info/download.html
2) Запускаем файлы в exe;
3) Созданные файлы с данными *.d помещаем в директорию с соответствующими 
им скриптами *.plt;
4) Запускаем gnuplot и последовательно открывем скрипты *.plt (эти файлы должны 
 быть в одной директории с соответствующими *.d-файлами), графики при желании 
сохраняем (эспортируем в pdf или другой формат). 

Работа программ должна быть понятна из комментариев, в противном 
случае — спрашивайте.
